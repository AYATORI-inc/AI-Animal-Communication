# 🦁 AIアニマル・コミュニケーション

> **あなたの想像力が、動物の運命（リアクション）を変える**

生成AIを活用した、予測不能なリアクションと視覚的な驚きを提供するインタラクティブな動物餌やりシミュレーションゲーム。

---

## ✨ 概要

OpenAIの最新モデル（GPT-4o および DALL-E 3）を活用し、ユーザーの入力に応じてリアルタイムで反応が変化する革新的なエンターテインメント体験を提供します。

従来の「正解が決まっているゲーム」とは異なり、生成AIの特性を活かした**無限のバリエーション**と**ビジュアルインパクト**により、ユーザーの没入感を最大化します。

---

## 🎯 コンセプト

- **無限のバリエーション**: 自由入力により、開発者さえ想定しなかったユニークな結果が生まれる
- **ビジュアルインパクト**: 自分の言葉が即座に「画像」として具現化される感動体験
- **シンプルな操作性**: 小学生～ファミリー層、デジタルに不慣れな高齢者層まで幅広く楽しめる

---

## 🎮 ゲームフロー

### フェーズ1: エンカウント（動物の出現）
- ランダム、または選択した動物（例：空腹のライオン）が登場
- 動物の性格に基づいた「おねだりセリフ」をAIが生成・表示

### フェーズ2: インタラクション（餌やり選択）
ユーザーは以下の2パターンからアクションを選択可能：

1. **クイック選択（4択ボタン）**
   - 正解（肉）、不正解（草）、ジョーク（タイヤ）、大穴（激辛料理）などをプリセット

2. **フリー入力（テキストボックス）**
   - ユーザーが自由な発想でアイテムを入力
   - 例：「お母さんの手作りハンバーグ」「魔法の薬」など

### フェーズ3: リアクション生成（本企画の肝）
- **テキスト生成**: 入力されたアイテムに対し、動物がどう感じるかをAIが判断し、感想を出力
- **画像生成**: 「そのアイテムを食べた（または接触した）結果の動物」をリアルタイムで画像生成
  - **成功例**: 満腹で幸せそうな顔
  - **失敗例**: 味が合わずに渋い顔、アイテムを頭に乗せている、などコミカルな描写

---

## 🛠️ 技術スタック

- **テキスト応答**: OpenAI GPT-4o（高速かつ自然な日本語対話）
- **画像生成**: OpenAI DALL-E 3（プロンプトへの忠実性が高い）
- **セーフティ**: OpenAI Moderation API + 独自NGワードフィルター

---

## 🔒 セーフティ機能

自由入力における不適切なワード（暴力・性表現など）に対し、以下の対策を実装：

- OpenAI Moderation APIによる自動検出
- 独自のNGワードフィルター
- 不快な画像の生成を未然に防止

---

## ⚡ パフォーマンス最適化

画像生成の待ち時間（十数秒）を体感的に短縮するため、以下の演出を実装：

- 生成中は「動物がモグモグ咀嚼しているアニメーション」を表示
- AIによる実況コメントを挟み、体感待ち時間を短縮
